
# Religions in ZooCity

This folder contains information on the major world religions as represented in ZooCity. Each religion is described in a child-friendly, system-agnostic way, with attention to real-world parallels and educational value.


## Reference List

- [Monopawism (The Law)](Monopawism.md) — Proxy for Judaism
- [Sunherdism](Sunherdism.md) — Proxy for Christianity (with major denominations)
- [Crescentism](Crescentism.md) — Proxy for Islam (with major denominations)
- [Riverism](Riverism.md) — Proxy for Hinduism (with major denominations)
- [Pathism](Pathism.md) — Proxy for Buddhism (with major denominations)
- [Atheism](Atheism.md) — Non-religion/worldview; largest and fastest-growing group in some districts, with faith in science and frequent criticism of organized religion

Each file includes:
- Overview and core beliefs
- Major factions/denominations
- Social frictions and issues
- Plot hooks for campaigns
